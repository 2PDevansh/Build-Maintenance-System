# 🏢 Building Maintenance System

![Python](https://img.shields.io/badge/Python-3.10-blue)
![PySpark](https://img.shields.io/badge/PySpark-Big%20Data-orange)
![Status](https://img.shields.io/badge/Status-Active-success)

A scalable **Building Maintenance Management System** designed to handle, track, and analyze maintenance requests efficiently using **Python and Apache Spark**.

---

## 🚀 Features

- 📝 Add maintenance requests  
- 📋 View all requests  
- 📊 Analyze data using **PySpark**  
- ⚡ Modular and scalable architecture  
- 🐳 Docker-ready (for deployment)  

---

## 🧠 Use Case

This system can be used in:
- 🏢 Residential buildings  
- 🏫 Campuses / Hostels  
- 🏬 Office spaces  

to manage maintenance workflows efficiently.

---

## 📊 Tech Stack

| Technology | Purpose |
|----------|--------|
| Python | Core application logic |
| PySpark | Data processing & analytics |
| CSV | Lightweight storage (HDFS-ready design) |
